## Criação de Váriavel

<p align="justify">Variável só acessada dentro da própria classe, vem por padrão com modificador private</p>

```js

[TIPO DA VARIAVEL][NOME DA VARIAVEL] = [VALOR];

```

## Tipo de Váriaveis

- int
- double
- string
- bool
- float
- char

## Sequencia de escape

- \n	 Nova linha
- \t	 Tabulação
- \b	 Backspace
- \"	 Aspa dupla
- \'	 Aspa simples
- \?	 Ponto de interrogação
- \\	 Barra invertida

}
